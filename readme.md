# 詳細
    本リポジトリはLaravelでWEBスクレイピングするテストを行うものである。(ライブラリはGoutteを利用)
    
    ただし、DBを利用して保管したり、404チェックを行っていたり、作者の目的により余分な機能が付与されておりますので、サンプルを参照する方は、下記を参照ください。
    https://github.com/shigyamamoto/LaravelSample01_Scraping
